Proyectos en los que he participado

Mascota Virtual

Empleando Python como lenguaje principal se realizó una máquina de estado finito en forma
de mascota virtual para que se manifestaran los distintos estados de la máquina.
Los estados se representaban mediante las acciones de la mascota, es decir si la mascota
movia su cola y se ponia feliz, entonces ese ya un primer estado y cada vez que cambiaba su
estado de animo, se decía que cambiaba el estado de máquina finita.


Simulador de un sistema de aeropuerto

Usando el famosísimo C++ se simularon en una consola todas las acciones de un sistema de aeropuerto:

  • Registrar clientes.

  • Eliminar clientes.

  • Modificar clientes.

  • Buscar clientes.

  • Imprimir información del vuelo.

Además se emplearon animaciones para simular el proceso de abordo, vuelo y aterrizaje. Fue un proyecto
muy curioso que a más de una persona si le generó muchos problemas por el limitado tiempo y la complejidad del sistema.


Conexión de una Base datos con Java

Es el proyecto más reciente que he hecho y se trató de hacer una
conexión de una base de datos a tráves de Java y para estó se utilizaron los progrmas de
PostgreSQL y de NetBeans (es el que contiene Java). Lo que se hizo fue la creación de una base
de datos del Covid-19 con sus respectivos contenidos y luego se conectó a una interfaz contenida en NetBeans,
en la cual se deberian ejecutar las siguientes acciones:

  • Registrar pacientes.

  • Eliminar pacientes.

  • Modificar pacientes.

  • Buscar pacientes.

  • Imprimir información del paciente.

Por supuesto, dicha base tenia muchos datos, entonces se crearon varias interfaces
para dividir la información y hacer una conexión más eficaz.
